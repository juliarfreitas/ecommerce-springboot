# ecommerce-springboot

API REST para sistema simplificado de e-commerce utilizando Spring Boot.

## Tecnologias

* Java 17
* Spring Boot
* Spring Data JPA
* H2 Database
* Lombok
* Swagger / OpenAPI

## Funcionalidades

* CRUD de categorias
* Relacionamentos JPA
* Repositories
* API REST
* Documentação Swagger

## Como executar

```bash
./mvnw spring-boot:run
```

## URLs

Swagger:
http://localhost:8080/swagger-ui.html

H2 Console:
http://localhost:8080/h2-console
